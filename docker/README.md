**Dependencies**
- CUDA: 10.2
- ROS: melodic
- Python: 3.8.11
- Pytorch: 1.6

**Building docker image**
```
    $ source build.sh
```

**Run the container**
```
    $ source docker_run.sh
    #Docker $ source environment.sh
```
**Join the container**
```
    $ source docker_join.sh
    Docker $ source environment.sh
```
